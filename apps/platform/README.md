# Argo CD Applications (plataforma)

Tras el bootstrap con `scripts/windows/install-phase2.ps1` (o `scripts/linux/install-phase2.sh`), puedes registrar aquí Applications para que Argo CD gestione upgrades de:

- metrics-server
- sealed-secrets
- aws-load-balancer-controller
- argocd (self-managed)

En la Fase 4 se añadirán Applications para `langfuse` y `mcp-server`.
